# Portfolio Offgrid

A simple personal portfolio website built with plain HTML, CSS, and JavaScript.

## Features

- Intro hero section with floating tech badges
- Experience, skills, and education modal sections
- Contact links
- Resume download button in the Work Experience modal

## Project Structure

- `index.html`: Page markup and content
- `styles.css`: Visual styles and layout
- `main.js`: Scroll-based animation and modal scroll-lock logic
- `images/`: Static assets (profile image and resume)

## Run Locally

1. Open the project folder.
2. Open `index.html` in your browser.

No build step or dependencies are required.
